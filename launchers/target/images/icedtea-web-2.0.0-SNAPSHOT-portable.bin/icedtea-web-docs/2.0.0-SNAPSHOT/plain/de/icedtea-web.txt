  ***  icedtea-web 2.0.0-SNAPSHOT  ***  
NAME
    Mit icedtea-web  wird ein Browser Plug-in als freie Software angeboten, mit dem Applets ausgeführt werden können, sowie eine Implementierung von „Web
    Start“, ursprünglich auf dem NetX-Projekt basierend.
    
    NetX ermöglicht das Herunterladen, die Zwischenspeicherung und die Ausführung von Applets und Anwendungen in einer sicheren Sandkastenumgebung
    (Standardeinstellung). Bei nachfolgenden Starts eines Applets wird die jeweils neueste Version automatisch heruntergeladen. Mit dem Befehl „itw-settings“
    können unter anderem Aktualisierungs- sowie Sicherheitseinstellungen gesetzt werden.
    „icedtea-web“ enthält ebenfalls ein Plug-in für „Applets aktivieren (http://www.java.com/en/download/testjava.jsp)“ in Webbrowsern.
    - Die Namen sowie E-Mail Adressen der an diesem Projekt Mitwirkenden wurden in der im Stammverzeichnis von IcedTea-Web befindenden Datei AUTHORS hinterlegt.
    - Eine vollständige Ausfertigung der GPLv2 Lizenz dieses Projekts wurde in der im Stammverzeichnis von IcedTea-Web befindenden Datei COPYING hinterlegt.
    - Neuigkeiten über die Veröffentlichungen dieses Projekts können der im Stammverzeichnis von IcedTea-Web befindenden Datei NEWS entnommen werden.


BESCHREIBUNG
    Merkmale von NetX:
    Modularität                          - Einfaches Hinzufügen von JNLP-Merkmalen zu einer Anwendung.
    Reduzierter Arbeitsspeicherverbrauch - Starten von Programmen in einer gemeinsamen JVM
    Schnellstart                         - Anwendungen aus einem Zwischenspeicher für schnelles Starten ausführen
    Sicherheit                           - Eine beliebige Anwendung in einer Sandkastenumgebung ausführen oder deren Tätigkeit protokollieren
    Automatische Aktualisierung          - Anwendungen ohne zusätzlichen Code automatisch aktualisieren
    Bereitstellung im Netzwerk           - Ohne Installer im Internet bereitstellen
