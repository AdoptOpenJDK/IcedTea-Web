    icedtea-web poskytuje svobodný zásuvný modul pro webový prohlížeč, který spouští aplety napsané v programovacím jazyce Java, a implementaci technologie Java Web Start, původně založené na projektu NetX.

    NetX umožňuje pro aplety a aplikace Java jejich stahování po síti, zapisování do vyrovnávací paměti a (ve výchozím stavu) spouštění v bezpečném izolovaném prostoru (sandbox). Následná spuštění apletu automaticky stáhnou nejnovější verzi. Nastavení aktualizací a zabezpečení lze nastavit (mimo jiné) pomocí příkazu itw-settings.
    icedtea-web také obsahuje zásuvný modul, který     zapne aplety Java ve webových prohlížečích.

    • POPIS

    Hlavní vlastnosti NetX:

  • Modulární - Můžete snadno dodávat aplikacím možnosti protokolu JNLP.
  • Šetří paměť - Můžete spouštět programy ve sdílených prostředích JVM.
  • Rychlý start - Urychlení startu aplikací spouštěním z vyrovnávací paměti.
  • Zabezpečení - Všechny aplikace můžete spouštět v izolovaném prostoru a protokolovat jejich aktivity.
  • Automatické aktualizace - Aplikace se můžou automaticky aktualizovat bez speciálního kódu.
  • Nasazení po síti - Nasazujte aplikace přes internet, nikoliv pomocí instalátorů.
  • Otevřený zdrojový kód - GNU Lesser General Public License.
